# One-page links (mobile-first)

## Что внутри
- `index.html` — страница
- `css/styles.css` — стили (mobile-first)
- `js/main.js` — минимальный JS
- `assets/` — положи сюда фон:
  - `background.jpg` (основная картинка)
- `icons/` — SVG иконки (уже сгенерированы)

## Как запустить
Самый простой вариант:
- открыть `index.html` двойным кликом в браузере

Если хочешь локальный сервер:
- `python -m http.server 8080`
- открыть `http://localhost:8080`

## Что поменять
В `index.html` замени ссылки:
- `tel:+3720000000`
- `https://t.me/your_username`
- `viber://chat?number=%2B3720000000`
- `https://wa.me/3720000000`
